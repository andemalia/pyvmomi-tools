pyvmomi-tools

Additional community developed python packages to help you work with pyvmomi and other closely related VMware API

Download this project as a .zip file Download this project as a tar.gz file

Welcome to pyVmomi-tools

The project consists of classes and methods developed to make community-samples easier to work with. The majority of these tools started life as part of a sample and then were cleaned up, revised, and promoted to a reusable tool.

Work In Progress

This project is very much a work in progress. It will be built by your collaboration and ultimately belongs to the Python community that is interested in developing software with VMware APIs.
